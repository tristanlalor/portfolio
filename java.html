Intelligent Battleship
Tristan Lalor
This was my first java app as well as my first time writing code for related objects. You can find the files on github, and the final product – a standalone application for Mac OS – on itch.io.
Project files: link to the github repository
Packaged download: link to itch.io page
I also got really into the graphic design (making the moving water, boats, etc), so I threw in some easter eggs to change the skin. The most difficult parts were figuring out how to package all of the GUI elements (JavaFX) into a standalone app. I ended up using a beta version of Jpackage along with Jlink. I wrote the intelligence to beat a strategic player ~50% of the time. Can you win?
