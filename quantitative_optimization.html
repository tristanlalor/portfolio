Quantitative Optimization Modeling
Below are 4 models that consistent of different asset data. These models assess the standard deviation of each security based on daily return data, create correlation and covariance matrices, and calculate portfolio expected return and variance / standard deviation based on varying weights of each asset. By defining constraints as a non-linear optimization model, we are able to solve for the optimal mix of securities to either maximize return given a minimum portfolio standard deviation, or minimize risk given a required return.
